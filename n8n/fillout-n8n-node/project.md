Built and published two community n8n nodes for the Fillout form platform — a trigger node that fires on new form submissions via webhook, and an action node for reading form data. Used by the n8n community for no-code/low-code form automation workflows.

Handles webhook registration and cleanup automatically on node activation/deactivation, and supports all Fillout form events.
