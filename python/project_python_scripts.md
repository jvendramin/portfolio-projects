A set of standalone Python projects built for learning and demonstration:

- **Expense Tracker** — Desktop GUI app with PostgreSQL storage and bar chart summaries
- **GitHub Stats Visualizer** — Fetches GitHub API data to report on languages, top repos, and contribution activity
- **Pokédex** — Terminal Pokédex using PokeAPI with stat comparisons
- **Space Tracker** — Live SpaceX launch data + ISS position via Open Notify API
- **Trivia Game** — CLI trivia with Open Trivia Database API, category/difficulty selection, and score tracking
- **Weather Dashboard** — 5-day forecast terminal dashboard with temperature trend charts
- **Country Explorer** — REST Countries API lookup with side-by-side country comparisons
